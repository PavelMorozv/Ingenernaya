Метод обределяющий минимальное количество дейтсвий для достижения цели

Аргументы
nЧисло от которого будут считаться действия (шаги) к цели "m"
minusДействие вычитания. Обычно используется "1"
kДействие деления. Может использоваться любое число, но не более число "n"
mЧисло "цель" до которого необходимо выполнять действия вычитания и деления
Возвращает
При удачном выполнении программмы возвращает количество шагов и выдает MessageBox с подробным решением и ответом
